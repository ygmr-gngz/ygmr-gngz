## 🚀 Ne Yapıyorum?

Büyük Dil Modelleri (LLM), **Retrieval-Augmented Generation (RAG)** ve modern backend mimarileriyle güçlendirilmiş, **üretim odaklı yapay zekâ sistemleri** geliştiriyorum.

Odak noktam; doküman işleme ve vektör aramadan, API dağıtımına ve konuşma arayüzlerine kadar **uçtan uca yapay zekâ çözümleri** tasarlamak. LLM yeteneklerini klasik makine öğrenmesi teknikleriyle birleştirerek **açıklanabilir, ölçeklenebilir ve güvenilir** uygulamalar üretmekten keyif alıyorum.

```python
yagmur = {
    "uzmanlik": ["LLM Uygulamaları", "RAG Mimarileri", "Konuşma Tabanlı AI"],
    "yaklasim": "Hibrit AI → LLM + Klasik Makine Öğrenmesi",
    "hedef":    "Fikirden üretime: gerçek problemlere gerçek çözümler 🎯"
}
```

- 🧠 Üretim kalitesinde **LLM uygulamaları** geliştiriyorum
- 🔎 **RAG mimarileri** tasarlıyor ve dağıtıyorum
- 💬 **Konuşma tabanlı AI sistemleri** (chatbot, asistan) inşa ediyorum
- ⚗️ LLM + ML'i birleştiren **hibrit yapay zekâ çözümleri** üretiyorum
- 🔌 Ölçeklenebilir **REST API'ler ve backend servisleri** yazıyorum
- 🐳 **Docker ve CI/CD** ile yapay zekâ sistemlerini canlıya alıyorum
